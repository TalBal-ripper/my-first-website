# HTML, CSS та JavaScript Демонстрація

Це простий навчальний проєкт для
демонстрації базового використання HTML, CSS та JavaScript.

На сторінці є блок та кнопка. Коли користувач натискає кнопку, текст у
блоці змінюється, а також змінюється його колір.

## Функціонал

-   Відображення текстового блоку
-   Кнопка для взаємодії з елементом сторінки
-   Зміна тексту за допомогою JavaScript
-   Зміна кольору блоку

## Використані технології

-   HTML
-   CSS
-   JavaScript

## Запуск проєкту

1.  Відкрийте файл index.html у будь-якому браузері.

або

2.  Просто двічі натисніть на файл `index.html`.

Після цього відкриється сторінка з демонстрацією роботи HTML, CSS та
JavaScript.
